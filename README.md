# スライドエディタ - Next.js版

HTMLベースのスライドエディタです。A4横向きスライドの作成・編集が可能で、リアルタイムプレビュー、画像挿入、自動保存などの機能を備えています。

## ✨ なぜこのスライドエディタを選ぶべきか

### 🚀 完全にクライアントサイドで動作
このスライドエディタは**アプリケーションサーバーやデータベースサーバーが不要**で、すべての処理がブラウザ上で完結します。データはローカルストレージに保存されるため、**外部サーバーにデータを送信することは一切ありません**。機密情報を含むスライドも安心して作成できます。

### 📦 静的ファイルとしてどこでも使える
`npm run export` で静的ファイルを生成すれば、**既存のWebサーバー（Nginx、Apacheなど）に配置するだけで即座に利用可能**です。アプリケーションサーバーは不要で、CDNや社内サーバー、さらにはローカルでWebサーバーを起動するだけでも動作します。シンプルな静的ファイルホスティングサービスでも利用できます。

### 🔒 組織内で完全にコントロール
- **データはすべてローカル**: クラウドサービスに依存しません
- **カスタマイズ自由**: ソースコードを自由に改変可能
- **ライセンス**: MITライセンスで商用利用も可能
- **オフライン対応**: インターネット接続がなくても動作

### 💰 コストゼロ
- サーバー費用不要
- クラウドサービス料金不要
- ライセンス費用不要
- 完全無料で組織内の全員が利用可能

### 🎯 すぐに始められる
複雑な設定は不要です。静的ファイルを配置するだけで、すぐにスライド作成を開始できます。社内のプレゼンテーション、資料作成、教育コンテンツ制作など、様々な用途で活用してください。

## 🎯 主な機能

### 基本機能
- **📝 HTMLエディタ**: リアルタイムプレビュー付きのHTMLエディタ
- **👀 プレビュー**: スライドのリアルタイムプレビュー
- **📊 ステータスバー**: スライド数、文字数、エラー/警告数の表示

### スライド管理
- **➕ スライド追加**: テンプレートから新しいスライドを追加
- **🗑️ スライド削除**: 不要なスライドを削除
- **📋 スライド複製**: 既存のスライドを複製
- **🔄 スライド順序変更**: ドラッグ&ドロップでスライドの順序を変更
- **📄 ページ番号自動更新**: スライドの追加・削除時に自動でページ番号を更新

### 画像機能
- **🖼️ 画像挿入**: ファイル選択・URL入力・ドラッグ&ドロップ対応
- **📁 画像管理**: 保存された画像の一覧表示と削除
- **🔄 Base64変換**: 外部画像をBase64形式に変換して埋め込み
- **📐 2分割レイアウト**: 画像挿入時に自動で左右2分割レイアウト

### データ管理
- **💾 自動保存**: LocalStorageによる自動保存（30秒間隔）
- **🔄 復元**: 前回の編集内容を復元
- **📋 HTMLコピー**: エディタの内容をクリップボードにコピー（画像も含む）
- **📥 クリップボードから読み込み**: クリップボードのHTMLをエディタに読み込み
- **↶↷ アンドゥ・リドゥ**: 編集履歴の元に戻す/やり直し（Ctrl+Z / Ctrl+Y）

### エディタ機能
- **🔍 検索・置換**: テキストの検索と置換（Ctrl+F）
- **⚠️ エラー検出・警告表示**: HTML構文エラーやCSSクラス未定義などの検出
- **📏 行番号表示**: エディタに行番号を表示
- **🎨 シンタックスハイライト**: （今後実装予定）

### 表示機能
- **🔗 別ウィンドウでプレビュー**: 新しいウィンドウでプレビューを表示
- **📐 パネルリサイズ**: エディタとプレビューのサイズをドラッグで調整

## 🚀 セットアップ

### 必要な環境
- Node.js 18以上
- npm または yarn

### インストール

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。

## 📦 ビルド

```bash
# プロダクションビルド
npm run build

# 静的エクスポート（Nginx環境で使用する場合）
npm run export
```

出力は `out/` ディレクトリに生成されます。

## ⌨️ キーボードショートカット

- `Ctrl+S` / `Cmd+S`: HTMLコピー
- `Ctrl+N` / `Cmd+N`: スライド追加
- `Ctrl+I` / `Cmd+I`: 画像挿入
- `Ctrl+R` / `Cmd+R`: 復元
- `Ctrl+O` / `Cmd+O`: 別ウィンドウでプレビュー
- `Ctrl+K` / `Cmd+K`: エディタクリア
- `Ctrl+F` / `Cmd+F`: 検索・置換
- `Ctrl+Z` / `Cmd+Z`: 元に戻す
- `Ctrl+Y` / `Cmd+Y`: やり直す
- `ESC`: メニューを閉じる
